### 🤖 自动化发布工作流

- **📄 文件**: `.github/workflows/table-auto-publish.yml`
- **功能**: 完整的表格批量发布自动化流程
- **特性**:
  - ⏰ 定时自动发布（北京时间 9:00, 15:00, 21:00）
  - 🔄 手动触发支持（支持参数配置）

  - 🚀 批量发布执行
  - 📝 自动状态更新和提交
  - 📈 详细发布报告生成

4. **发布执行**

## 🎯 使用方式

### 自动定时发布

```yaml
# 自动执行时间（UTC）
- cron: '0 1,7,13 * * *'

# 对应北京时间
- 9:00 (办公时间开始)
- 15:00 (午后时间)  
- 21:00 (晚间时间)
```

### 手动触发参数

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `table_file` | 字符串 | 指定表格文件 | `articles.csv` |
| `platforms` | 字符串 | 指定发布平台 | `devto,hashnode` |
| `draft_mode` | 布尔值 | 草稿模式 | `true/false` |
| `force_all` | 布尔值 | 强制发布所有 | `true/false` |

每次执行都会生成详细报告：

1. **执行信息**

   - 触发方式（定时/手动）
   - 配置参数

2. **配置状态**
   - 平台API密钥状态
   - 权限检查结果

3. **文件处理**
   - 发现的表格文件

4. **发布结果**
   - 成功/失败统计
